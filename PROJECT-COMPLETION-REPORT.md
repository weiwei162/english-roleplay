## 📝 经验总结

### 成功经验
1. **模块化设计** - pi-agent 与主服务分离，易于扩展
2. **文档先行** - 完善的文档降低使用门槛
3. **Docker 部署** - 一键部署减少运维成本
4. **真实 LLM** - 使用真实模型提升体验

### 遇到的挑战
1. **字幕格式解析** - 火山引擎二进制字幕需要手动解析
2. **工具调用延迟** - 多次工具调用增加响应时间
3. **成本控制** - 需要在质量和成本间平衡
4. **儿童内容安全** - 需要严格的内容过滤

### 解决方案
1. **字幕解析** - 实现 subv 格式解析器
2. **延迟优化** - 使用流式输出、预加载
3. **成本控制** - 使用 gpt-4o-mini 等经济模型
4. **内容安全** - 系统提示词 + 工具过滤
