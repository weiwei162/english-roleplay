## 修改的文件

### 1. `server/index-join-ai.js`
```javascript
// 之前
const frontendPath = path.join(__dirname, '..');

// 之后
const frontendPath = path.join(__dirname, '..', 'frontend');
```

### 2. `server/Dockerfile.main`
- 简化了 COPY 路径，直接使用项目根目录作为 build context
- 前端文件从 `frontend/` 复制
- 启动命令改为 `node server/index-join-ai.js`

### 3. `frontend/index.html`
- 移除了 CSS 链接（改为在 main.js 中导入）
- 移除了所有 `<script>` 标签
- 添加了 Vite 入口：`<script type="module" src="/main.js"></script>`

### 4. 新增文件
- `frontend/main.js` - Vite 入口文件，导入所有 JS 和 CSS
- `frontend/package.json` - NPM 配置，包含 Vite
- `frontend/vite.config.js` - Vite 配置，包含 API 代理
- `frontend/.gitignore` - 前端忽略文件
- `.dockerignore` - 更新，忽略 frontend/node_modules

## 使用方法

### 开发模式（需要安装依赖）
```bash
# 安装前端依赖
cd frontend
npm install

# 启动 Vite 开发服务器（端口 5173，代理 /api 到后端）
npm run dev

# 在另一个终端启动后端
cd server
npm install
node index-join-ai.js
```

### 生产构建
```bash
# 构建前端到 server/dist
cd frontend
npm run build

# 构建 Docker
cd ..
docker-compose build
```

### Docker 部署
```bash
docker-compose up -d
```
