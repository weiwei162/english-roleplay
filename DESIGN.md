## 产品概述

### 产品名称
**English Friend - 英语好朋友** 🐾

### 产品定位
为 3-6 岁儿童设计的**沉浸式对话型**英语启蒙应用，通过角色扮演和场景互动，让孩子在玩耍中自然习得英语。

### 核心价值
- ❌ 不是答题游戏
- ❌ 不是传统聊天机器人
- ✅ 是**会动的动画片式学习伙伴**
- ✅ 在**场景对话**中自然学习
- ✅ 像**朋友一样建立关系**

### 使用场景
- 🏠 家庭英语启蒙
- 📱 平板/手机随时学习
- 🖥️ 电脑大屏体验
- 👨‍👩‍👧 亲子互动学习

## 核心设计理念

### 1. 画布驱动，非聊天框驱动
```
传统聊天应用：
┌─────────────────┐
│ 对话记录        │
│ 对话记录        │
│ 对话记录        │
│ [输入框]        │
└─────────────────┘

English Friend：
┌─────────────────┐
│    🦁    🦒     │
│       👩‍🏫      │ ← 角色在画布上
│   💬           │
│    🐘    🦓     │
│ [互动区]        │
└─────────────────┘
```

### 2. 角色是"活"的
- 🚶 可以在画布上移动
- 💃 有丰富的表情和动作
- 🎯 会指向正在讨论的内容
- 👆 可以点击互动

### 3. 开放式对话
- 没有标准答案
- 角色主动发起话题
- 理解并回应孩子的话
- 记住孩子的信息（名字、喜好等）

### 4. 视觉化学习
- 说到什么显示什么
- 图片/视频/emoji 动态出现
- 点击可放大查看
- 多感官刺激

## 详细功能说明

### 1. 角色系统 🎭

#### 1.1 可选角色
| 角色 | 形象 | 性格 | 声音类型 | 适合场景 |
|------|------|------|----------|----------|
| Miss Emma | 👩‍🏫 | 温柔耐心 | 女声 | 日常教学 |
| Tommy | 👦 | 活泼好动 | 童声 | 游戏玩耍 |
| Lily | 👧 | 热情开朗 | 女声 | 情感交流 |
| Coach Mike | 👨‍🦱 | 积极阳光 | 男声 | 运动互动 |
| Grandma Rose | 👵 | 慈祥温暖 | 老年声 | 关怀对话 |

#### 1.2 角色动画
| 动画名称 | 触发条件 | 视觉效果 | 音效 |
|----------|----------|----------|------|
| `floating` | 待机 | 轻微上下浮动 | - |
| `walking` | 移动时 | 左右摇摆 | 脚步声 |
| `talking` | 说话时 | 身体起伏 | 语音 |
| `pointing` | 指向内容 | 转身面向内容 | - |
| `jumping` | 开心/回到中心 | 向上跳跃 | 弹出音 |
| `waving` | 打招呼 | 挥手动作 | - |
| `clapping` | 被点击/鼓励 | 拍手动作 | 笑声 |
| `excited` | 兴奋时 | 抖动 | 欢呼声 |
| `thinking` | 思考时 | 歪头 | - |

#### 1.3 角色移动
- **智能定位**：角色自动走到讨论内容旁边
- **路径规划**：根据内容位置决定移动方向
- **朝向控制**：面向内容（左/右）
- **过渡动画**：0.8s 缓动效果

#### 1.4 角色互动
- **点击回应**：点击角色会笑并随机回应
- **提示气泡**：悬停显示"Click me! 👋"
- **情感反馈**：根据互动显示不同情绪

### 3. 对话系统 💬

#### 3.1 对话流程
```
1. 角色主动发起对话
   ↓
2. 显示对话气泡（英文 + 中文）
   ↓
3. 显示相关媒体内容（动物/物品）
   ↓
4. 角色移动到内容旁边
   ↓
5. 孩子回应（语音/点击选项）
   ↓
6. 角色理解并回应
   ↓
7. 进入下一个对话
```

#### 3.2 对话类型

**开放式问题**
- "What's your name?"
- "How are you feeling today?"
- "What did you eat?"

**引导式问题**
- "Can you say LION?"
- "What color is this?"

**情感交流**
- "You make me happy!"
- "I'm proud of you!"

#### 3.3 智能理解

**关键词匹配规则**
```javascript
{
  keywords: ['name', 'call me', "i'm"],
  responses: [
    "Nice to meet you, {name}!",
    "{name}! I love your name!"
  ],
  memory: 'name'  // 记住孩子的名字
}
```

**支持的关键词类别**
- 👤 名字相关
- 🎂 年龄相关
- 🍕 食物相关
- 🎨 颜色相关
- 🐾 宠物相关
- 🧸 玩具相关
- 😊 情感表达
- ✅ 肯定/否定

#### 3.4 记忆系统

**记忆内容**
- 孩子的名字
- 年龄
- 喜欢的颜色
- 喜欢的食物
- 喜欢的玩具
- 宠物信息
- 上次访问时间

**记忆使用**
- 下次见面时称呼名字
- 根据喜好推荐话题

### 5. 音效系统 🎵

#### 5.1 音效列表
| 音效名称 | 触发场景 | 频率 | 时长 |
|----------|----------|------|------|
| `step` | 角色移动 | 每步一次 | 0.1s |
| `pop` | 内容显示/点击 | 每次出现 | 0.15s |
| `cheer` | 完成场景 | 每次完成 | 0.3s |
| `laugh` | 点击角色 | 每次点击 | 0.5s |
| `correct` | 答对/鼓励 | 正确时 | 0.2s |

#### 5.2 技术实现
```javascript
// Web Audio API 合成音效
function playSound(type) {
  const oscillator = audioContext.createOscillator();
  const gainNode = audioContext.createGain();
  
  oscillator.connect(gainNode);
  gainNode.connect(audioContext.destination);
  
  switch(type) {
    case 'step':
      oscillator.frequency.value = 200;
      oscillator.type = 'sine';
      break;
    case 'pop':
      oscillator.frequency.value = 600;
      oscillator.type = 'sine';
      break;
    // ... 其他音效
  }
  
  oscillator.start();
  oscillator.stop();
}
```

### 6. 语音系统 🎤

#### 6.1 语音识别（输入）
- **技术**：Web Speech API
- **语言**：英语（en-US）
- **模式**：单次识别
- **浏览器支持**：Chrome/Edge/Safari

**使用流程**
1. 按住"按住说话"按钮
2. 开始录音（按钮变绿色，脉冲动画）
3. 松开按钮，自动识别
4. 显示识别结果
5. 角色理解并回应

#### 6.2 语音合成（输出）
- **技术**：Web Speech API
- **语言**：英语（en-US）
- **语速**：0.75（慢速，适合儿童）
- **音调**：1.1（稍高，更友好）

**语音控制**
- 朗读对话气泡内容
- 可点击 🔊 按钮重播
- 角色说话时有口型动画

#### 6.3 快速回复
当语音识别不可用时，提供快速回复选项：
- "Yes!"
- "No!"
- "I like it!"
- "Tell me more!"
- "What's next?"

## 用户体验设计

### 视觉设计原则
1. **大字体**：适合儿童阅读
2. **高对比度**：清晰可见
3. **鲜艳色彩**：吸引注意力
4. **圆角设计**：柔和友好
5. **动画反馈**：即时响应

### 交互设计原则
1. **简单直观**：无需教学
2. **即时反馈**：每次操作都有响应
3. **容错设计**：没有"错误"，只有鼓励
4. **多模态**：语音 + 点击 + 视觉

### 无障碍设计
- 🎨 高对比度配色
- 🔊 视觉 + 听觉双重反馈
- 👆 大按钮，易点击
- 📱 响应式，适配各种屏幕

### 性能优化
- 💾 本地存储，减少网络请求
- 🎬 CSS 动画，硬件加速
- 📦 按需加载资源
- ⚡ 首屏加载 < 2 秒
